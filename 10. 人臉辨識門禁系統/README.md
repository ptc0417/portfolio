# 10. 人臉辨識門禁系統
###### 課程：
嵌入式系統設計
###### 使用語言：
Arduino C/C++
###### 說明：
本專題使用基於 ESP-32S 模組所設計的核心開發板 NodeMCU-32S 作為系統中樞，搭配 Huskylens AI 辨識鏡頭執行人臉辨識、完成人臉註冊並比對，
最後加上按鈕、繼電器、RFID等硬體模組構成一個「人臉辨識門禁系統」。此系統可使用註冊過的人臉進行解鎖，亦可使用 RFID Card 進行解鎖動作。
